## REGRA ÚNICA
**Antes de executar qualquer coisa — definir o resultado final esperado.**

```
1. O QUE deve existir quando terminar?
2. Como VERIFICO que existe?
3. O que acontece se FALHAR?
4. Quem é NOTIFICADO?
```

Só depois executar.
