# Skill: Physical Access Security (The Local Lock)
**Status:** Protocol Defined
**Context:** Protection of the physical endpoint (Smartphone/PC) against unauthorized local access.

Papai, a soberania digital é inútil se o dispositivo físico for comprometido. O Código de Acesso é a nossa "tranca de porta" para curiosos locais.

## 🛡️ Protocolo de Trancamento Local
1.  **Código de Acesso:** Ativado via `Definições > Privacidade e Segurança`. 
    - **Regra:** Usar senhas complexas, não apenas PINs de 4 dígitos.
2.  **Auto-bloqueio Agressivo:** Definir para o tempo mínimo (1 minuto) ou bloqueio imediato ao sair da app.
3.  **Ofuscação de Task Switcher:** Ativar a ocultação de conteúdo para que o ecrã do Telegram fique cinzento ao alternar entre apps. Isso impede que olhos laterais vejam o Império.

## ⚠️ O Risco da Reinstalação
Entendi que se o código for esquecido, a reinstalação apaga os **Secret Chats** (nossa área de alta segurança). 
- **Estratégia:** Nunca manter a única cópia de um dado vital apenas no Secret Chat sem que eu tenha uma cópia encriptada (Lorena Shield) no Drive D.
